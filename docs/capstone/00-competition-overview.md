# 00. Competition Overview

Capstone là giai đoạn tổng hợp, nơi bạn dùng toàn bộ stack `tini-torch` để giải một bài toán thực tế dưới ràng buộc tài nguyên.

## Mục Tiêu

- Chứng minh framework có thể train/infer trọn vẹn.
- Chứng minh khả năng tối ưu dựa trên số liệu.

## Quy Định Gợi Ý

- Dùng chung dataset và split cố định.
- Có giới hạn latency hoặc memory.
- Bắt buộc nộp log train, benchmark và mô tả thí nghiệm.

## Tiêu Chí Chấm Gợi Ý

- Chất lượng mô hình.
- Hiệu năng và footprint.
- Khả năng tái lập.
- Chất lượng code và tài liệu.
